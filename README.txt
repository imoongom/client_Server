How to compile instruction

    # in source folder
	- make file : 'run'
	 	to make executable file.

	- client execute:  ./client [FILENAME with extesion]
		and than input priority between 1 to 5 after instruction. (1 is high priority)

	- server execute: ./server
		


     # example of client side with file name
	./client novel.txt
	./client bear.jpg
	./client book.txt
